For Qurado, I worked in a minor consulting capacticy to demonstrate how Figma, Adobe XD, and tools likeNotion would increase website traffic and engagement.

During the consulting phase, I worked with the client to create a few new concepts on the original design, and made a few sugggetions on how to improve the user experience.

Sitemap

My first task was to create a sitemap for the new design. I used the existing sitemap as a starting point, and made a few changes to the structure to make it more user-friendly.

Branding

We worked on creating a few new branding concepts for the new design. We wanted to create a more modern and sleek look for the app, so we experimented with different color schemes and typography.

Workflows

We worked on checkout flows and payment flows for the new design. We wanted to make the checkout process as smooth and easy as possible for our users, so we tested a few different designs and made a few changes to the flow.

Deployment

Utilizing the same color pallet and typography, we created a modernized design and asthetic for the site. This included a hero video to introduce the brand.
