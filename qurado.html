Link

Live Site

Overview

Qurado works directly with the best local producers throughout Spain to deliver fresh products. For this engagement, I worked in a specialized consulting capacity to demonstrate how tools like Figma, Adobe XD, and Notion could increase website traffic and user engagement.

During the consulting phase, I worked with the client to create new concepts on the original design and made targeted suggestions to improve the user experience, specifically focusing on the checkout flow and mobile optimization.

The Problem

The original platform suffered from low conversion rates due to a confusing checkout process, lack of branding consistency, and a poor mobile experience.

The Solution

We implemented a streamlined checkout flow, modernized the branding to reflect the premium quality of the products, and optimized the sitemap for better navigation.

The Process

Analysis, Branding, and Optimization

Sitemap Wireframe
01

Sitemap & Structure

My first task was to create a sitemap for the new design. I used the existing sitemap as a starting point and made key changes to the structure to make it more user-friendly, reducing the number of clicks required to reach products.

Branding System Design
02

Branding & Identity

We worked on creating new branding concepts for the redesign. We wanted to create a more modern and sleek look for the app, so we experimented with different color schemes and typography to align with the fresh, organic nature of the products.

Workflows Checkout
03

Workflows & Checkout

We focused heavily on checkout flows and payment processing. We wanted to make the checkout process as smooth and easy as possible for our users to reduce cart abandonment, so we tested a few different designs and simplified the flow.

Final Design Deployment
04

Deployment

Utilizing the defined color palette and typography, we created a modernized design and aesthetic for the site. This included a hero video introduction and a fully responsive layout for tablets and mobile devices.
